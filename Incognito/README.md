Discord fires analytics on virtually every interaction. Incognito blocks all of it: tracking, telemetry, usage profiling, error reporting, and process scanning. Also hides what you've read and when you're typing, disables idle detection, spoofs fingerprints, strips URL tracking, anonymises uploads, hides client mods, and more. All features can be toggled from the settings panel.
